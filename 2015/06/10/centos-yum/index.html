CentOS yum 使用笔记

2015.06.10 nextkitt Linux  热度 ℃

yum [options] [command] [package …]

常用options

  • -y 当安装过程提示选择全部为”yes”
  • -q 运行过程不打印日志
  • -v 运行过程打印详细日志

安装软件包

yum install package-name

如：

yum install httpd-tools

查询是否安装过某软件包

yum list |grep package-name

如：

yum list |grep httpd-tools

查询某软件包中包含哪些文件

使用yum-utils中的repoquery命令，首先安装yum-utils

yum install yum-utils

查询： 

repoquery --list package-name

如：

repoquery --list httpd-tools
/usr/bin/ab
/usr/bin/htdbm
/usr/bin/htdigest
/usr/bin/htpasswd
/usr/bin/logresolve
/usr/share/doc/httpd-tools-2.2.15
/usr/share/doc/httpd-tools-2.2.15/LICENSE
/usr/share/man/man1/ab.1.gz
/usr/share/man/man1/htdbm.1.gz
/usr/share/man/man1/htdigest.1.gz
/usr/share/man/man1/htpasswd.1.gz
/usr/share/man/man1/logresolve.1.gz
