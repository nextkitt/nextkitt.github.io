挖井人要失业了

2014.08.08 nextkitt 杂记  热度 ℃

挖井人即将失业。

最近调研了 ucloud、 又拍云 和 七牛云存储 这三家的产品。

ucloud

  1. 云主机: 和linode类似，不过技术差一大截。
  2. 带宽
  3. 数据库: mysql, percona
  4. memory: redis, memcached
  5. 负载均衡: 4层负载均衡
  6. CDN： 按流量计费，价格透明
  7. 监控
  8. 按购买产品和流量收费
  9. 不需要签合同

又拍云

  1. 一系列图片解决方案：上传，下载，水印，尺寸，清晰度，旋转等等
  2. CDN
  3. 按存储空间用量和流量收费
  4. 不需要签合同

七牛云存储

  1. 存储：图片，普通文件
  2. 音视频解决方案：上传，转码，下载，防盗链，加速等等
  3. CDN
  4. 按调用接口次数、存储空间用量和流量收费
  5. 需要签合同

（又拍云和七牛云存储应该是竞争对手。）

总结

他们的特点是：

  1. 为客户提供一整套的解决方案
  2. 价格透明（可以到他们网站上去查，都是“明码标价”）
  3. 减少前期资金投入：这三家的服务可以按小时或天或月或年收费。
  4. 不是特点的特点：这三家的CDN都是使用网宿的服务（网宿这么diao）。

使得客户减少了一部分人力成本和时间成本的投入，降低成本。减少了哪些人力：

  1. 部分运维
  2. 部分后台开发人员

假如你要创办一个小型互联网公司，业务涉及图片及视频。

  1. 你要找机房、测试、买带宽，找供应商买设备，谈价格，磁盘换了找供应商返修，机器挂了找机房等等，人力上一个人搞不下来。时间上估计两个星期也下不来。讨价还价一般运维也干不来。
  2. 要维护这些设备，搞架构，搞监控，搞灾备。
  3. 后台：搞上传，音视频转码，下载，播放，图片切图等等。

使用现有的产品，你可以省去这些，也就意味者原先做这些工作的人要去其他公司另谋高就。

打个不恰当的比方：

20年前自来水还没有普及到农村，几乎每家院子里都有一口井（压水机），也能经常看见每家盖新房子的时候都会请挖井师傅们挖口井。而现在自来水的普及，导致那些挖井人另谋他路了。（我老家是山东农村，3年前县政府统一装铺了自来水。）
