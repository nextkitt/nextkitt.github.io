如何导出浏览器chrome和firefox的firebug调试信息

1 chrome

按F12或在页面空白处按鼠标右键选择审查元素进入调试模式, 点击network，然后右键选择Save as HAR with Content保存到文件就可以了。

详细见：export chrom developer data

2 firefox

详细见：export firebug data

3 HAR

HAR就是HTTP Archive。W3C规定了HAR的格式。
