shell - time

2013.12.19 nextkitt Linux  热度 ℃

time实例

time用于打印一条命令的执行时间。如：

$ wc -l /etc/passwd
43 /etc/passwd

real    0m0.001s
user    0m0.000s
sys     0m0.000s

time的real、user、sys的含义

  1. real指从程序运行开始到程序结束的总时间。
  2. user指程序本身，以及程序调用用户态库函数的执行时间。
  3. sys指程序直接或简介调用系统调用函数的执行时间。

如果将time的输出结果重定向到文件

time的输出为stderr，重定向的方法是运行子shell或将time cmd写到一个shell脚本中然后对脚本重定向。运行子shell方法`(cmds;)

#使用子shell
$ (time wc -l /etc/passwd) 2> log
43 /etc/passwd
$ cat log 

real    0m0.002s
user    0m0.000s
sys 0m0.000s
$ (time wc -l /etc/passwd) &> log
$ cat log 
43 /etc/passwd

real    0m0.001s
user    0m0.000s
sys 0m0.000s

#使用shell脚本
$ echo "time wc -l /etc/passwd" > t.sh
$ cat t.sh 
$ cat t.sh 
time wc -l /etc/passwd
$ bash t.sh &> log
$ cat log 
43 /etc/passwd

real    0m0.001s
user    0m0.000s
sys 0m0.000s
