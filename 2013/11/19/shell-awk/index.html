shell - awk

2013.11.19 nextkitt Linux  热度 ℃

awk 常用变量

  1. NR 全局行数(第二个文件第一行接着第一个文件尾行顺序计数)
  2. FNR 当前文件自身的行数
  3. NF 总字段数
  4. FS 字段分隔符
  5. OFS 输出字段分隔符
  6. ORS 输出记录分隔符

awk 基本操作

awk多个分隔符

统计Web Server条日志的时分秒，

echo '127.0.0.1 - - [22/May/2013:10:22:57 +0800] "GET / HTTP/1.1" 200 502' | awk -F'[: ]' '{print $5":"$6":"$7}'

awk调用shell命令

第一种方式是 system("cmd")，第二种方式是"cmd"|getline var； system()的缺点是,无法获取命令的输出，它只返回cmd的退出码；而getline可以获取命令的输出。对 date 命令的调用：

echo "05 Sep 2012 18:25:40"| awk '{sprintf("date -d \"%s\" +%%s", $0) | getline tm;print tm}'
echo "05 Sep 2012 18:25:40"| awk '{cmd=sprintf("date -d \"%s\" +%%s", $0); cmd| getline tm;print tm; close(cmd"|getlined)}'

这里的 close必须，因为awk使用一个内部管道就占用一个文件描述符，当使用描述符超过上线时就会报错，所以这里必须close；而且close的参数要和指定的shell命令完全一样，否则awk视为两个不同的管道，通常的做法是：先将shell命令赋值给一个临时变量，执行这个临时变量,然后close它。

#使用 awk 把第一列时间字符串字段 转换为时间戳输出
echo "10:00:00" | awk '{system("/bin/date -d "$1" +%s" "%Y-%m-%d" "%H:%M:%S")}'

#使用 awk 把第一列时间字符串字段 转换为时间戳输出
echo "10:00:00" | awk '{print "/bin/date -d "$1" +%s" "%Y-%m-%d" "%H:%M:%S"|"/bin/bash"}'

awk引用shell变量

#打印出 /etc/passwd 第一列为 "root" 的行
un="root"; awk -F':' '{if($1==t_un){print $0}}' t_un=${un} /etc/passwd

#打印出 /etc/passwd 第一列为 "root" 的行
un="root"; ubash="/bin/bash"; awk -F':' -v t_un=${un} -v t_ubash=${ubash} '{if($1==t_un && $NF==t_ubash){print $0}}' /etc/passwd
un="root"; awk -F':' '{if($1 ~ /^'"$un"'$/){print $0}}' t_un=${un} /etc/passwd

awk数组的用法

awk数组针对字符串的用法

#统计 /etc/passwd 那些用户使用同样的shell
awk -F':' '{arr[$7]=arr[$7]" "$1;}END{for (k in arr) {print k, arr[k]}}' /etc/passwd

awk数组针对数字的用法

#统计 /etc/passwd 第3列的和
awk -F':' '{sum+=$3}END{print sum}' /etc/passwd

awk匹配

awk行匹配

#打印出 /etc/passwd 的10到20行
cat -n /etc/passwd | awk -F':' 'NR>=10 && NR<=20 {print $0}'

#打印出 /etc/passwd 的10到20行中包含字符串 "ftp" 的行
cat -n /etc/passwd | awk -F':' 'NR>=10 && NR<=20 && /ftp/ {print $0}'

#打印出 /etc/passwd 的10到20行中 以/bin/sh结尾的行
cat -n /etc/passwd | awk -F':' 'NR>=10 && NR<=20 && /\/bin\/bash$/ {print $0}'

awk比较字符串和数字

cat /etc/passwd | awk -F':' '{if($3>"1000"){print $3}}'  #比较字符串
cat /etc/passwd | awk -F':' '{if($3>1000){print $3}}'  #比较数字
